MCD

Cuando tenemos un grupo de n números y queremos saber si existe algún número que sea capaz de dividirlos a todos sin dejar residuo podemos hacer uso de varios métodos para poder encontrar el MCD que por sus siglas es el Máximo Común Divisor. El MCD es el mayor número capaz de dividir exactamente a un grupo de números.

Parte de un división

Antes de enseñarte cómo es que funciona el algoritmo de euclides, debes saber cuales son las partes de una división ya que es indispensable para poder comprender este tema de buena manera.

  • Dividendo (D)

    Este es el número al cual vamos a dividir.

  • Divisor (d)

    Es la cantidad de partes en las que vamos a dividir al dividendo.

  • Cociente (q)

    Es el resultado de la operación, específicamente es el tamaño de cada parte en las que se dividió el dividendo

  • Residuo (r)

    En las divisiones exactas es el sobrante de la operación, es decir, cuando el divisor ya no puede seguir dividiendo en partes iguales.
